# Apple Website

A modern, polished landing page built with React and Vite, inspired by the clean aesthetic and product-focused storytelling of Apple.

## ✨ Features

- Responsive, modern UI
- Smooth section-based layout for showcasing products and brand story
- Fast development workflow with Vite
- Clean React component structure
- Easy local preview and deployment

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- CSS
- pnpm

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/ezedinmoh/Apple-website.git
   cd Apple-website
   ```

2. Install dependencies
   ```bash
   pnpm install
   ```

3. Start the development server
   ```bash
   pnpm dev
   ```

4. Open your browser and visit
   ```text
   http://localhost:5173/
   ```

## 📁 Project Structure

```text
src/
  components/
  css/
  main.jsx

```

## 🧪 Build for Production

```bash
pnpm build
```
